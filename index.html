HTML

  • the head element contains information about the webpage
  • the body element represents the visable content of the webpage
  • HTML attributes are always defined in the begininng element, never the closing element
  • doctype html :declartion telling the computer that it is a html document
  • meta :defines metadata about html document, always in the head element
  • h :header, numbers after the h signify the order of significance
  • html lang=en :tells the browser the language is english
  • p :paragraph or block of text
  • ul :unordered list
  • ol :ordered list
  • li :list item
  • br :line of empty space or break
  • img :image
  • src :location of image
  • a :anchor (creates links to same webpage)
  • href :points to the URL for the link

CSS

  • There are three ways to style a webpage using CSS:
  • Internal CSS style sheet
  • Internal CSS style sheet can be used in the head of HTML to style everything in the same category
  • inline CSS is used on individual lines to style on their own
  • External CSS style sheet
  • External CSS style sheet is in a seperate file, allowing for easier changes to design as a whole (uses a link placed the HTML head)
  • CSS rule: property(color):value(blue)
  • declartion block example= div {color:blue; text-align: justify}
  • A margin indicates how much space we want around the outside of the element
  • A padding indicates how much space we want around the content inside an element

Git

  • cd (name of location) :move to file
  • cd ..:move back a file
  • ls :show file's content
  • code . :open VScode
  • git checkout :changes branches
  • git status :check which branch currently on
  • git add :add modification to current working branch
  • -a :add all changes
  • git checkout -b -name :creates new branch

JavaScript

  • JavaScript should be linked in the body element, whereas CSS is linked in the head
  • JavaScript is read top to bottom and order matters. You must be intentional about setting up code in a JS file
  • A variable is a named container that allows us to store our data in our code.
  • Control flow is the order in which a computer executes code in the script.
  • String : this is a sequence of text known as a string - let myVariable = 'Bob'; to signify that the value is a string, enclose it in a single quote mark
  • Numbers dont have quotes - let myVariable = 10;
  • Boolean : this is a true/false value - let myVariable = true; the words true and false are special keywords that don't require quote marks
  • Array : this is a structure that allows you to store multiple values in a single refernce - let myVariable = [1, 'Bob', 'Steve', 10]
  • Object : this can be anything. Everything in JavaScript is an object and can be stored in a vafriable.
  • Comments : comments are snippets of text that can be added along with the code. The browser ignores text marked as comments - /* everything here is a comment */
  • Operator : an operator is a mathematical symbol used that produces a result based on two values (or variables)
  • Conditionals : conditionals are code structures used to test if an expression remains true or not. (e.g. if____ else)
