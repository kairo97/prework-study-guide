HTML

  • the head element contains information about the webpage
  • the body element represents the visable content of the webpage
  • HTML attributes are always defined in the begininng element, never the closing element
  • doctype html :declartion telling the computer that it is a html document
  • meta :defines metadata about html document, always in the head element
  • h :header, numbers after the h signify the order of significance
  • html lang=en :tells the browser the language is english
  • p :paragraph or block of text
  • ul :unordered list
  • ol :ordered list
  • li :list item
  • br :line of empty space or break
  • img :image
  • src :location of image
  • a :anchor (creates links to same webpage)
  • href :points to the URL for the link
